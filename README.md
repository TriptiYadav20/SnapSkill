# SnapSkill 

SnapSkill is a modern skill-sharing and learning platform designed to help users **showcase skills, discover talent, and connect for learning or collaboration**. The platform focuses on quick skill discovery, profile-based showcasing, and easy interaction between learners and skilled individuals.



##  Project Description

SnapSkill aims to bridge the gap between **people who want to learn** and **people who have skills to offer**. Instead of long resumes or complex portfolios, SnapSkill enables users to present their abilities in a simple, structured way so others can quickly understand what they do.

It can be used by:

* Students showcasing technical or creative skills
* Freelancers displaying services
* Learners searching for mentors or peers
* Communities promoting skill exchange



##  Key Features

*  **User Authentication** – Secure login and registration
*  **Skill-Based Profiles** – Add and manage skills easily
*  **Skill Discovery** – Find people based on skills
*  **Connect & Collaborate** – Reach out to skilled users
*  **Responsive UI** – Works across devices
*  **Fast & Lightweight** – Optimized for performance

##  How to Run the Project


1. **Navigate to the project directory**

cd SnapSkill

2. **Install dependencies**

npm install


3. **Start the application**

npm start


4. Open your browser and visit:

http://localhost:3000
